# GenAI Prompt Engineering – LangChain Prompt Templates

This repository contains the implementation of a **Mini Prompt Engine** built using LangChain PromptTemplate and ChatPromptTemplate. The assignment focuses on designing dynamic and reusable prompt systems instead of hardcoded prompts.


## 📌 Objective

The objective of this assignment is to build a dynamic and reusable prompt system using LangChain PromptTemplate. The goal is to design structured prompt pipelines that accept user inputs and generate flexible prompts for different use cases. This assignment demonstrates real-world prompt engineering techniques used in Generative AI applications.
